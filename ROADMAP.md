# Roadmap

Questa roadmap raccoglie le principali attività previste per migliorare Aulab Post come progetto portfolio.

## Completato

- Struttura Laravel MVC.
- Autenticazione utenti.
- Gestione articoli.
- Ruoli e middleware autorizzativi.
- Dashboard Admin, Revisor e Owner.
- Gestione candidature tramite sezione "Lavora con noi".
- README professionale.
- Changelog.
- Licenza MIT.

## In corso

- Miglioramento della documentazione tecnica.
- Preparazione screenshot per il README.
- Organizzazione del repository per uso portfolio.

## Prossimi miglioramenti

### Documentazione

- Aggiungere screenshot di homepage, login, registrazione e dashboard.
- Aggiungere una sezione con credenziali demo locali.
- Documentare il flusso di revisione degli articoli.
- Documentare il flusso delle candidature.

### Codice

- Valutare il refactoring del controller `YourController` con un nome più descrittivo.
- Raggruppare i controller per area funzionale.
- Aggiungere test automatici per rotte principali e middleware.
- Creare seeders demo per utenti, ruoli, categorie e articoli.

### UI/UX

- Uniformare lo stile delle dashboard.
- Migliorare la leggibilità delle pagine di gestione.
- Ottimizzare la visualizzazione mobile.

### Deploy

- Preparare una possibile demo online.
- Documentare configurazione produzione.
- Aggiungere istruzioni per ambiente locale e produzione.
