# Changelog

Tutte le modifiche principali del progetto saranno documentate in questo file.

## [1.0.0] - 2026-07-09

### Aggiunto

- Piattaforma editoriale basata su Laravel.
- Autenticazione e registrazione utenti tramite Laravel Fortify.
- Sistema di ruoli: Admin, Revisor, Writer, Owner e utente registrato.
- Creazione, visualizzazione, modifica ed eliminazione articoli.
- Filtri per categoria e autore.
- Dashboard revisore per accettare o rifiutare articoli.
- Dashboard admin per gestione utenti, ruoli e candidature.
- Dashboard owner per gestione avanzata delle candidature.
- Sezione pubblica "Lavora con noi".
- Middleware dedicati per il controllo degli accessi.
- README professionale per presentazione portfolio.
- Licenza MIT.
